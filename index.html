Core Principles

🔒 Strict Typing

Every script should use strict mode. 

--!strict

⚡ Functional First

Prefer pure functions and composition whenever practical.

📝 Semicolons

Statements should end with semicolons. 

print("Hello");
workspace.Gravity = 196.2;

Naming Conventions

Services PascalCase InventoryService
Variables camelCase playerData
Private Functions camelCase calculateReward()
Public Functions PascalCase GetCoins()
Classes PascalCase PlayerProfile
Data Tables snake_case player_inventory

const over local

H Luau Style prefers const for values intended not to be reassigned. 

const Players = game:GetService("Players");
const ReplicatedStorage = game:GetService("ReplicatedStorage");

Example Service

--!strict

const CoinService = {};
CoinService.__index = CoinService;

type PlayerData = {
    coins: number;
};

const player_data: {[Player]: PlayerData} = {};

function calculateReward(
    player: Player
): number
    return 10;
end;

function CoinService.GiveCoins(
    player: Player,
    amount: number
): ()
    const data = player_data[player];

    if not data then
        return;
    end;

    data.coins += amount;
end;

function CoinService.GetCoins(
    player: Player
): number
    const data = player_data[player];

    if not data then
        return 0;
    end;

    return data.coins;
end;

return CoinService;

Philosophy

"Consistency beats preference."

H Luau Style values predictable code, strict typing, and reducing unnecessary style debates within teams.
