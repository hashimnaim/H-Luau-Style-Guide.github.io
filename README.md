# H Luau Style Guide

H Luau Style is a Roblox Luau coding convention focused on consistency and maintainability.

## Principles

- Use `--!strict` in every script.
- Prefer functional programming.
- Use semicolons.
- Services use PascalCase.
- Variables use camelCase.
- Private functions use camelCase.
- Public functions use PascalCase.
- OOP classes use PascalCase.
- Data tables use snake_case.
- Prefer `const` for values intended not to be reassigned.

## Philosophy

Consistency over personal preference.
